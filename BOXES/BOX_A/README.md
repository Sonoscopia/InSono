# DESCRIPTION
BOX A - the smallest box 

#OBJECTS
- 2 Small Gongs

#PARTS
- Arduino Uno
- 1 Motor(12V)
- 1 LED
- 1 HC-SR04 Ultrasonic sensor

#NOTES
Arduino must be powered by a 12V adapter (through jack) in order to get the 12V motor moving.
